About Decora

A personal ML demo exploring AI-powered interior design.

Decora takes a photo of your room, uses Google Gemini's vision API to extract its style, colour palette, and mood, then searches the IKEA catalogue and scores each product for fit. The top matches are ranked by score and returned with price and rating data.

Technology

Google Gemini vision API for room analysis and product scoring. IKEA catalogue API for real-time product search.

Stack

Python + Flask backend. Deployed on Vercel. Built as a portfolio demonstration of multimodal AI.
